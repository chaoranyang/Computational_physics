# 三体问题的数值模拟：稳定解与混沌解

## 1. 物理背景

三体问题是经典力学中的著名难题，研究三个质量在相互万有引力作用下的运动。与二体问题不同，三体问题不存在一般的解析解，其运动轨迹可以呈现规则或混沌行为。本实验采用四阶 Runge‑Kutta 方法，分别模拟一个稳定解（Euler 共线解）和一个混沌解，并验证稳定解在特定时刻的共线性。

系统由三个质点组成，质量分别为 $m_ i$（$i = 0, 1, 2$），位置矢量为 $\vec{r}_ i = (x_ i, y_ i)$，速度矢量为 $\vec{v}_ i = (v_ {x i}, v_ {y i})$。牛顿引力定律给出第 $i$ 个质点所受第 $j$ 个质点的引力为

$$
\vec{F}_ {ij} = - \frac{G m_ i m_ j}{|\vec{r}_ {ij}|^3}   \vec{r}_ {ij}
$$

其中 $\vec{r}_ {ij} = \vec{r}_ i - \vec{r}_ j$，$|\vec{r}_ {ij}|$ 为距离。取 $G = 1$，所有物理量无量纲化。

每个质点的运动方程（二阶常微分方程组）为

$$
\ddot{\vec{r}}_ i = \sum_ {j \neq i} \frac{m_ j}{|\vec{r}_ {ij}|^3}   (\vec{r}_ j - \vec{r}_ i)
$$

该方程组可改写为一阶形式，引入速度 $\vec{v}_ i = \dot{\vec{r}}_ i$，则

$$
\dot{\vec{r}}_ i = \vec{v}_ i, \qquad \dot{\vec{v}}_ i = \sum_ {j \neq i} \frac{m_ j}{|\vec{r}_ {ij}|^3} (\vec{r}_ j - \vec{r}_ i)
$$

---

## 2. 数值方法：四阶 Runge‑Kutta (RK4)

采用经典四阶 Runge‑Kutta 方法对上述一阶系统进行时间离散。设状态向量为

$$
\vec{Y} = (x_0, y_0, x_1, y_1, x_2, y_2, v_ {x0}, v_ {y0}, v_ {x1}, v_ {y1}, v_ {x2}, v_ {y2})^\top
$$

则系统可写为 $\dot{\vec{Y}} = \vec{f}(t, \vec{Y})$。RK4 的每一步更新为

$$
\begin{aligned}
\vec{k}_1 &= \Delta t   \vec{f}(t_ n, \vec{Y}_ n), \\
\vec{k}_2 &= \Delta t   \vec{f}(t_ n + \frac{\Delta t}{2}, \vec{Y}_ n + \frac{\vec{k}_1}{2}), \\
\vec{k}_3 &= \Delta t   \vec{f}(t_ n + \frac{\Delta t}{2}, \vec{Y}_ n + \frac{\vec{k}_2}{2}), \\
\vec{k}_4 &= \Delta t   \vec{f}(t_ n + \Delta t, \vec{Y}_ n + \vec{k}_3), \\
\vec{Y}_ {n+1} &= \vec{Y}_ n + \frac{1}{6}(\vec{k}_1 + 2\vec{k}_2 + 2\vec{k}_3 + \vec{k}_4)
\end{aligned}
$$

该方法具有四阶精度，适用于长期积分。

## 3. 任务 (a)：Euler 稳定解（共线解）

**设定**：三个质量分别为 $m_0 = 3$，$m_1 = 1$，$m_2 = 1$。初始位置（均位于 y 轴上）为

$$
\vec{r}_0 = (0, 1), \quad \vec{r}_1 = (0, -0.83299), \quad \vec{r}_2 = (0, -2.16700)
$$

初始速度为

$$
\vec{v}_0 = (-0.63034, 0), \quad \vec{v}_1 = (0.52507, 0), \quad \vec{v}_2 = (1.36596, 0)
$$

取时间步长 $\Delta t = 0.01$，总时间 $T = 7$。使用 RK4 求解，并绘制三个质点的轨迹。

**结果**：三个质点在运动中保持近似共线，整体绕质心旋转，形成稳定的“旋臂”图案（类似左图 Fig.1）。

---

## 4. 任务 (b)：共线性数值验证

从任务 (a) 的数值解中，提取 $T = 1, 2, 4, 6$ 时刻三个质点的位置坐标。利用三角形面积公式判断共线性：若三点 $(x_0, y_0)$、$(x_1, y_1)$、$(x_2, y_2)$ 构成的面积

$$
S = \frac{1}{2} \left| x_0(y_1 - y_2) + x_1(y_2 - y_0) + x_2(y_0 - y_1) \right|
$$

小于给定容差（如 $10^{-3}$），则认为三点共线。输出每个时刻的面积及是否共线的结论，以此证明 Euler 解在指定时间切片下确实保持共线。

## 5. 任务 (c)：混沌解

**设定**：三个质量相等 $m_0 = m_1 = m_2 = 1$。初始位置和速度为

$$
\vec{r}_0 = (3, 0), \quad \vec{r}_1 = (-1.5, 2.59), \quad \vec{r}_2 = (-0.6, -2.65)
$$

$$
\vec{v}_0 = (0.1, 0.2), \quad \vec{v}_1 = (-0.2, -0.1), \quad \vec{v}_2 = (0.1, -0.1)
$$

时间步长 $\Delta t = 0.01$，总时间 $T = 20$。同样采用 RK4 积分，绘制三个质点的轨道。

**结果**：轨道呈现非周期、不规则缠绕，且对初始条件敏感，表现出典型的混沌特征（类似右图 Fig.1）。与稳定解形成鲜明对比。

## 6. 代码实现与关键函数

代码使用 Python 3 及 NumPy、Matplotlib 库。核心类 `TBP` 封装了三体问题的所有功能：

- `__init__(m, pos, vel)`：初始化质量、初始位置和速度，构建状态向量 $\vec{Y}_0$。
- `g_acc(r1, r2, m_val)`：计算单个引力加速度（$m_ val$ 对 $r_1$ 的引力）。
- `deriv(t, y)`：计算状态向量对时间的导数，即 $\vec{f}(t, \vec{Y})$。
- `rk4(dt, T)`：执行 RK4 积分，返回时间数组和完整解矩阵。
- `get_traj(sol)`：从解矩阵中提取三个质点的轨迹 $(x_ i(t), y_ i(t))$。
- `chk_col(pos, tol=1e-3)`：给定三个位置，计算三角形面积并判断是否共线。

辅助函数 `plt_orb(traj, m, title, filename)` 用于绘制轨道，并标记起点（圆点）和终点（方块）。

**执行流程**：
1. 创建 (a) 的实例，调用 `rk4` 求解，绘制轨道图并保存为 `fig_a.png`。
2. 在指定时间点提取位置，调用 `chk_col` 打印共线性验证结果。
3. 创建 (c) 的实例，求解并绘制轨道图，保存为 `fig_c.png`。

## 7. 结果讨论

- **稳定解**（Euler 解）：三个质点始终保持在一条直线上，整体匀速旋转，轨道为闭合或准周期曲线，符合理论预期。
- **共线性验证**：数值结果确认在选取的时间切片上面积几乎为零，证明了 Euler 解的共线性质。
- **混沌解**：轨道无明显周期，多次交叉且不重复，体现了三体问题的复杂性。RK4 方法在 $\Delta t = 0.01$ 下能较好地捕捉细节，长时间积分仍保持能量守恒（可通过监测总能量验证，但本实验未要求）。
