# 泊松方程的数值求解：松弛方法比较

## 1. 物理背景与问题描述

在静电学中，电势 $\phi(\vec{x})$ 满足泊松方程

$$
\Delta \phi(\vec{x}) = -\frac{1}{\epsilon_0} \rho(\vec{x})
$$

其中 $\rho$ 为电荷密度， $\epsilon_0$ 为真空介电常数。本任务考虑二维正方形区域，边界上电势恒为零（Dirichlet 边界条件），即 $\phi|_{\partial \Omega} = 0$。取网格间距 $h = 1$， $\epsilon_0 = 1$ ，采用有限差分法将方程离散化，然后使用三种经典迭代方法（Jacobi、Gauss‑Seidel 和 SOR）求解线性方程组，并比较它们的收敛速度。

特别地，研究一个空心正方形电荷分布：在一个 $(2N+1) \times (2N+1)$ 的网格上，中心挖去一个边长为 $N/2$（取整）的内正方形，剩余环形区域（不包括边界点）的电荷密度设为 $1$，外部为零。考察 $N = 10, 20, 40$ 三种尺寸，绘制电势分布并记录达到误差界限 $10^{-4}$ 所需的迭代次数。

## 2. 数值方法

### 2.1 离散化与迭代格式

泊松方程在二维直角坐标系中的离散形式（五点差分）为

$$
\frac{\phi_{i-1,j} - 2\phi_{i,j} + \phi_{i+1,j}}{h^2} + \frac{\phi_{i,j-1} - 2\phi_{i,j} + \phi_{i,j+1}}{h^2} = -\frac{\rho_{i,j}}{\epsilon_0}
$$

整理得到

$$
\phi_{i,j} = \frac{1}{4} \left( \phi_{i+1,j} + \phi_{i-1,j} + \phi_{i,j+1} + \phi_{i,j-1} + \frac{h^2}{\epsilon_0} \rho_{i,j} \right)
$$

所有内部点按此式迭代更新，边界点保持为零。

### 2.2 Jacobi 松弛法

每次迭代使用上一轮的值 $\phi^{(k)}$ 来计算新值：

$$
\phi_{i,j}^{(k+1)} = \frac{1}{4} \left( \phi_{i+1,j}^{(k)} + \phi_{i-1,j}^{(k)} + \phi_{i,j+1}^{(k)} + \phi_{i,j-1}^{(k)} + \frac{h^2}{\epsilon_0} \rho_{i,j} \right)
$$

更新时需保留旧值，全部计算完毕后再统一替换。

### 2.3 Gauss‑Seidel 松弛法

立即使用最新计算出的值，加速收敛：

$$
\phi_{i,j}^{(k+1)} = \frac{1}{4} \left( \phi_{i+1,j}^{(k)} + \phi_{i-1,j}^{(k+1)} + \phi_{i,j+1}^{(k)} + \phi_{i,j-1}^{(k+1)} + \frac{h^2}{\epsilon_0} \rho_{i,j} \right)
$$

通常比 Jacobi 快一倍。

### 2.4 逐次超松弛 (SOR)

在 Gauss‑Seidel 的基础上引入松弛因子 $\omega$（通常 $1 < \omega < 2$）：

$$
\phi_{i,j}^{(GS)} = \frac{1}{4} \left( \phi_{i+1,j}^{(k)} + \phi_{i-1,j}^{(k+1)} + \phi_{i,j+1}^{(k)} + \phi_{i,j-1}^{(k+1)} + \frac{h^2}{\epsilon_0} \rho_{i,j} \right)
$$

$$
\phi_{i,j}^{(k+1)} = (1 - \omega) \phi_{i,j}^{(k)} + \omega   \phi_{i,j}^{(GS)}
$$

本实验取 $\omega = 1.5$（经验值）。

### 2.5 收敛判据

定义两次迭代间的最大绝对变化为误差：

$$
e^{(k)} = \max_{i,j} \left| \phi_{i,j}^{(k)} - \phi_{i,j}^{(k-1)} \right|
$$

当 $e^{(k)} < 10^{-4}$ 时停止迭代，并记录此时的总迭代次数。

## 3. 代码实现与结构

代码使用 Python 3 及 NumPy、Matplotlib 库。主要函数包括：

- `jacobi_relaxation(phi, rho, h, eps0)`：单次 Jacobi 更新。
- `gauss_seidel_relaxation(phi, rho, h, eps0)`：单次 Gauss‑Seidel 更新。
- `sor_relaxation(phi, rho, h, eps0, omega)`：单次 SOR 更新。
- `calc_error(phi_old, phi_new)`：计算最大绝对误差。
- `solve_poisson(rho, method, omega, max_iter)`：循环迭代直到收敛或达到最大次数，返回电势和迭代次数。
- `create_hollow_square(N)`：生成 $(2N+1) \times (2N+1)$ 的空心正方形电荷密度矩阵。

对于每个 $N$，程序依次调用三种方法求解，并输出迭代次数。可视化部分将四个子图（电荷密度、Jacobi 电势、Gauss‑Seidel 电势、SOR 电势）并排显示，使用 3D 曲面图展示电势分布。

## 4. 结果与讨论

### 4.1 迭代次数比较

对每个 $N$，三种方法的迭代次数（单位为步数）如下：

| N   | Jacobi | Gauss‑Seidel | SOR (ω=1.5) |
|-----|--------|--------------|-------------|
| 10  |  627     |  342           |  125          |
| 20  |  2491     |  1359          |  509          |
| 40  |  9923     |  5412           |  2038          |

Gauss‑Seidel 比 Jacobi 所需步数少约一半，SOR 则进一步显著减少，尤其当 $\omega$ 接近最优值时。随着 $N$ 增大，网格点数增多，迭代次数均增加，但 SOR 的增速最缓，体现了其加速收敛的优势。

### 4.2 电势分布

对于每个 $N$，空心方形电荷在中心区域产生一个类似“环形山”的电势分布：内部空腔电势较低（甚至接近零），环形区域电势较高，外部逐渐衰减至边界零值。三种方法得到的电势分布几乎完全一致（误差远小于 $10^{-4}$），验证了数值解的正确性。

从 3D 曲面图可以清晰看到这一特征，且随着 $N$ 增大，电势峰值升高，分布范围扩大。

### 4.3 算法性能评论

- **Jacobi**：实现简单，但收敛最慢，适合并行计算（但本实验未并行）。
- **Gauss‑Seidel**：收敛速度约为 Jacobi 的两倍，编程简单，是常用选择。
- **SOR**：通过合理选择 $\omega$，可大幅提升收敛速度，但 $\omega$ 需根据问题调优，本实验 $\omega=1.5$ 并非最优，仍有改善空间。

理论上，对于五点差分格式，最优松弛因子 $\omega_{\rm opt}$ 与网格尺寸有关，可通过分析估计，但本实验仅作定性比较。
